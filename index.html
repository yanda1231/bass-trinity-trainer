使い方ガイド

基本の流れ
キーとパターンを選ぶ → Space で開始/停止。
タイミングを計測したい人は Audio Input の AUDIO をON。
AUDIOがOFFでも、メトロノーム＋指板ガイドで練習できます。タイミング計測がいらない人は右上の♪（シンプルモード）で、メトロノームとガイドだけの表示になります。

1セットとは
開始音から上行 → 頂点で折り返し → 開始音に戻るまで。U-TURN＝頂点を2回弾くか1回か。

判定の読み方
＋＝LATE（遅れ）／−＝EARLY（先走り）／判定幅内＝JUST。
判定幅は Judge Window で変更できます。TIGHT ±10（プロ基準の目安）／NORMAL ±20／EASY ±35。
Miss＝弾かなかった音、Ghost＝予定にない余計なヒット。
各数字の意味は、マウスを重ねると説明が出ます。

AUDIOモードで判定するまで（3ステップ）
1. AUDIO をONにして入力デバイスを選ぶ
2. Auto Setup — 静かに2秒待ち、案内に沿ってE・A・D・G各開放弦を普段どおり弾くだけ。感度合わせはこれにお任せ
3. Loopback Calibration — 出力→入力をケーブル1本で直結し、音の「配達の時間」を自動測定。真剣に測る日は「開いたら較正1回」

スライダー（Sense / Refract / Gate / Advanced）は、Auto Setupで解決しないときだけ。役割はマウスオーバーで出ます。

メトロノームとお手本
Click＝クリックの鳴る位置（2&4／毎拍／8分ウラ／Custom）。
Guide＝正解の音のお手本（音量は Guide Volume）。聴きながら重ねて、慣れたら絞って卒業。
スピーカーアイコンで、クリックとお手本を1小節だけ試聴（OFFにしている音は鳴りません）。

困ったときは
・検出されない → まず Auto Setup。だめなら Boost でメーター7〜8割 → Sense を下げる
・「入力が過大です」の赤警告 → 入力ゲインを下げて「復帰」
・設定が迷子 → 「設定を初期値に戻す」（較正・履歴は残ります）。全部消すなら「全データを消去」

較正（キャリブレーション）って何？
パソコンの中を音が通るときは、目に見えない「配達の時間」がかかります。これを測って差し引かないと、ぴったり弾いたのに「遅い」と出ます。AUDIO入力の人は Loopback Calibration を使ってください（出力→入力をケーブル1本で直結し、テスト音16回で自動測定。画面のチェックリストに従えば安全です）。測った値は保存されますが、ブラウザを開き直すと数ミリ秒ずれることがあります——真剣に計測する日は「開いたら較正1回」。なお、リミッターはアプリ出力の天井であり、I/F内部のモニターループ発振は止められません。較正しなくてもメトロノームとガイドはそのまま使えます。

この画面は ESC か外側クリックでも閉じられます。
