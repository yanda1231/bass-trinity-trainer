# Bass Trinity Trainer 取扱説明書

ベースの練習アプリです。**メトロノーム＋指板の運指ガイド＋タイミング判定**が1画面に入っています。ブラウザで開くだけで動き、インストールは不要です。

- **パソコンのChromeまたはEdgeで開いてください**（スマホ・タブレットは今のところ未対応です。キー操作と音声入力がPC前提のため）
- アプリ本体：https://yanda1231.github.io/bass-trinity-trainer/
- 合言葉（パスワード）画面が出たら、配布者から聞いた合言葉を入れてください（一度通れば次からは聞かれません）

## クイックスタート

1. Chrome または Edge で上のURLを開く
2. キーとパターンを選び、**Space** でスタート／ストップ
3. メトロノームに合わせて練習。タイミング判定で遊ぶなら **Enter** が「弾く」ボタン（T1モード）
4. 楽器の音で判定したい人は下の「AUDIOモードと較正」へ

## シンプルモード ♪

右上の **♪ボタン** を押すと、計測系の画面が全部消えて**メトロノームと指板ガイドだけ**のすっきり表示になります。タイミング計測に興味がない人はこれだけでOK。もう一度押せば元に戻ります（設定は覚えています）。

## 判定の読み方

- **＋の数字＝LATE（拍より遅れて弾いた）／−の数字＝EARLY（先走った）**
- **Just**：ズレが判定幅（初期値±20ms）に収まった割合
- **Average**：全ヒットの平均ズレ。**Median**：ズレを並べたときの真ん中の値（1回の大外しに引きずられない、実力の中心値）
- **SD**：ばらつき。小さいほど毎回同じタイミングで弾けている。目安：10ms未満はかなり安定（プロの音楽家で4〜20ms程度と言われます）
- **Trend**：平均ズレの向き。判定幅の半分（最大±15ms）以内ならCENTER

## AUDIOモードと較正（キャリブレーション）

パソコンの中を音が通るとき、目に見えない「配達の時間」がかかります。メトロノームの音が鳴るまで、あなたの音がアプリに届くまで、それぞれ数十ミリ秒（1000分の数十秒）の遅れがあります。この遅れを測って差し引かないと、ぴったりに弾いたのに「遅い」と判定されてしまいます。

- **Loopback Calibration（本番用）**：オーディオインターフェースの出力と入力をケーブル1本で直結し、アプリがテスト音を16回鳴らして往復時間を正確に測ります。AUDIOモードで練習する人は最初にこれをやってください。画面のチェックリストに従えば安全に進められます
- **Test-only Calibration（お試し用）**：Enterキーで遊ぶだけの人向けの簡易補正です。マイクやケーブルは不要
- 測った値はブラウザに保存されますが、**ブラウザを開き直すと数ミリ秒ずれることがあります。真剣に計測する日は、開いたら較正1回、が合言葉です**
- 較正しなくてもメトロノームとガイドはそのまま使えます（シンプルモード♪ならそもそも較正不要）

## Onset Detection（音の立ち上がり検出）

AUDIOモードのとき、アプリはあなたが弾いた「音の立ち上がり（オンセット）」を聴き取って、メトロノームとのズレを測ります。3つのスライダーは、ぜんぶ「どの音を本物の1音と認めるか」を決める門番の設定です。

| スライダー | 役割 | こんなときに動かす |
|---|---|---|
| **Sense**（感度） | 門番の厳しさ | 弾いたのにカウントされない→下げる ／ 弾いてないのに反応（Ghost）→上げる |
| **Refract**（休止時間） | 1音認めたあと次を受け付けない時間 | 1音が2回数えられる→上げる ／ 速いフレーズを取りこぼす→下げる |
| **Gate**（無音ゲート） | これより小さい音は門前払い | ノイズや弦のこすれで誤反応→上げる（-60くらい） ／ 小さい音も拾いたい→下げる |

### Advanced（普段は触らなくていい）

音の分析エンジンの内部設定です。困っていないなら初期値のままが正解。

- **Frame**：音を分析する「窓」の大きさ。大きいほど音色の変化に気づきやすいが、反応がのんびりになる
- **Hop**：分析の頻度。小さいほど時間の精度が上がるが、パソコンの負担が増える
- **Median pre / post**：「直前の音量の流れ」と比べるための参照範囲。ノイズの多い環境で判定がふらつくときの調整用
- いじって分からなくなったら、設定リセットで全部初期値に戻せます（較正値は消えません）

## 困ったときは

| 症状 | 対処 |
|---|---|
| マイク許可のダイアログが出ない | アドレスバーの鍵アイコン→マイクを「許可」に |
| 入力メーターが動かない | 入力デバイスとChannel（通常1ch）の選択を確認 |
| メーターは動くのに検出されない | 入力ゲインを上げる（メーターのピークが7〜8割まで）。それでもだめならSenseを下げる |
| 「入力が過大です」の赤警告 | 入力ゲインを下げてから「復帰」ボタン |
| 判定がいつも同じ方向にズレる | ループバック較正をやり直す（開いたら較正1回） |
| 表示がおかしい・設定が迷子 | 設定リセット→再読み込み |
